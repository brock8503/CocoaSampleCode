ReadMe for AUPinkNoise
----------------------

AUPinkNoise project demonstrates how to build a Generator Audio Unit. As the name implies, it generates pink noise.

Sample Requirements
-------------------
This sample project requires:
	
	Mac OS X v10.9 or later
	Xcode 5.0 or later
